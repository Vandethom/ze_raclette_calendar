# Guide du calendrier — Ze Raclette

Bienvenue sur le calendrier de la guilde ! Ce guide t'explique tout ce que tu peux faire sur le site en moins de 5 minutes.

---

## Première connexion — choisir ton pseudo

À ta première visite, une fenêtre s'ouvre et te demande ton **pseudo en jeu**.

- Saisis exactement ton pseudo Dofus (celui que tes collègues reconnaissent) ainsi que ta classe en jeu
- Clique **Confirmer**
- Ce pseudo est retenu pour toutes tes prochaines visites

> Pour le changer plus tard, clique sur ton pseudo en haut à droite de la page.

---

## Le calendrier

### Naviguer

| Bouton | Action |
|--------|--------|
| `<` `>` | Semaine précédente / suivante |
| `Aujourd'hui` | Revenir à la semaine en cours |
| `Mois` | Passer en vue mensuelle |
| `Semaine` | Revenir en vue semaine (défaut) |
| `Jour` | Zoom sur une seule journée |

### Lire les événements

Chaque bloc coloré sur le calendrier est un événement proposé par un membre :

- **Bloc doré** — événement que tu as créé ou auquel tu participes
- **Bloc bleu** — événement d'un autre membre que tu n'as pas encore rejoint

Le texte affiché suit ce format :
```
Pseudo — Nom de l'événement (Niveau) · Rôle
```
*Exemple : Kohuky — Comte Harebourg (200) · Tank*

Les **événements longue durée** (concours, challenges…) apparaissent dans la **bande "Long"** en haut du calendrier et s'étirent sur plusieurs jours.

## Proposer un événement

1. **Clique sur un créneau** du calendrier (le jour et l'heure qui t'intéressent)
2. La fenêtre de création s'ouvre avec la date pré-remplie
3. Remplis les informations :

### Événement court (donjon, boss…)

| Champ | Obligatoire | Exemple |
|-------|-------------|---------|
| Nom de l'événement | Oui | Comte Harebourg |
| Date | Oui | 10/05/2026 |
| Heure de début | Oui | 18:00 |
| Heure de fin | Oui | 20:00 |
| Ton rôle | Non | Tank, Heal, Do Crit… |
| Places max | Non | 4 *(toi + 3 autres)* |
| Niveau | Non | 200 |
| Description | Non | "Stuff Do Crit recommandé" |

### Événement longue durée (concours, challenge…)

Active le **bouton "Événement sur plusieurs jours"** pour faire apparaître un champ "Date de fin" :

| Champ | Obligatoire | Exemple |
|-------|-------------|---------|
| Nom de l'événement | Oui | Concours de skins |
| Date de début | Oui | 10/05/2026 |
| Date de fin | Oui | 24/05/2026 |
| Ton rôle | Non | (libre) |
| Description | Non | "Voter dans #skins-vote" |

4. Clique **Créer l'événement**

L'événement apparaît immédiatement sur le calendrier de tous les membres connectés.

> **Astuce mobile** : si tu es sur téléphone, utilise le bouton **+ Proposer un événement** en bas de l'écran.

### Choisir ton rôle

Le champ **"Ton rôle"** est un menu déroulant avec les options suivantes :

| Rôle | Description |
|------|-------------|
| Tank | Protection, aggro |
| Heal | Soins |
| Ret PM | Retrait de PM |
| Do Pou | Dommages de zone / poussée |
| Do Crit | Dommages critiques |
| Autre (préciser) | Saisis ton propre rôle |

Si ton rôle ne figure pas dans la liste, sélectionne **"Autre (préciser)"** et tape ton rôle manuellement.

## Rejoindre un événement

1. Clique sur un événement bleu qui t'intéresse
2. La fiche de l'événement s'ouvre — tu y vois :
   - L'activité, la date et les horaires
   - Le rôle de l'organisateur (s'il en a précisé un)
   - Le nombre de places restantes
   - La liste des membres déjà inscrits avec leur rôle
3. Si des places sont disponibles, **choisis ton rôle** dans le menu déroulant (optionnel)
4. Clique **Rejoindre**
5. Ton pseudo et ton rôle apparaissent dans la liste — visible en temps réel pour tout le monde

> Si l'événement affiche **Complet**, toutes les places sont prises. Tu peux quand même consulter la fiche pour voir les détails.

---

## Quitter un événement

Tu t'es inscrit mais tu ne peux finalement plus venir ?

1. Clique sur l'événement (il est doré puisque tu y participes)
2. Clique **Quitter**
3. Ta place se libère pour un autre membre

---

## Supprimer un événement

Seul **l'organisateur** peut supprimer son événement.

1. Clique sur ton événement
2. Clique sur l'icône **poubelle** (en bas à droite de la fiche)
3. Un message de confirmation s'affiche — clique **Confirmer la suppression**

Tous les participants sont automatiquement retirés de l'événement.

---

## Temps réel

Le calendrier se met à jour **automatiquement** pour tout le monde.
Tu n'as pas besoin de rafraîchir la page pour voir les nouvelles inscriptions ou les nouveaux événements.

---

## Récapitulatif rapide

```
Créer un événement court      →  Cliquer sur un créneau → remplir le formulaire
Créer un événement long       →  Activer "Plusieurs jours" → choisir début et fin
Préciser son rôle             →  Menu "Ton rôle" dans le formulaire de création
Rejoindre                     →  Cliquer sur un événement bleu → "Rejoindre"
Quitter                       →  Cliquer sur l'événement doré → "Quitter"
Supprimer                     →  Cliquer sur son événement → icône poubelle
Changer de pseudo             →  Cliquer sur son pseudo en haut à droite
```
